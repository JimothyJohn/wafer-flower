# [Wafer Flower](https://jimothyjohn.github.io/wafer-flower/)

It's the seeds of an idea.

Not a set of programming instructions, but a set of build instructions.

Dimensions clearly understood, relationships alive and interconnected, numbers converted to text to numbers to vectors to topology from your brain onto the 3D printer...it's not a wafer flower, it's not AI slop either, it's not even hard. It's it.

## Really though, what is it?

It's an experiment in how to collaborate iwth AI in both the digital and physical world. Information is contextualized, digested understood, and converted into beautifully-printed reality.
